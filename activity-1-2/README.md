## Descripción
Este repositorio contiene el análisis de la relación entre el nivel de felicidad de distintos países y variables económicas y de bienestar, utilizando modelos de regresión lineal simple y múltiple.

El estudio se basa en datos del World Happiness Report y variables económicas (GDP), así como indicadores proxy de bienestar y salud mental (PROMIS).

## Archivos
- **A1.2 Regresión lineal JosueB 635654.ipynb**  
  Notebook con el desarrollo completo del análisis, explicación teórica, gráficas y conclusiones.

- **A1.2 Felicidad y GDP.csv**  
  Conjunto de datos original con felicidad y GDP.

- **A1.2_Felicidad_GDP_PROMIS.csv**  
  Conjunto de datos extendido con variables proxy de bienestar (PROMIS).

## Notas metodológicas
Debido a la falta de una base de datos pública y homogénea a nivel país para instrumentos clínicos como PHQ-9 y GAD-7, se utilizaron indicadores proxy de bienestar ampliamente aceptados en estudios internacionales.
